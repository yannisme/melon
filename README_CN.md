# Melon

一款清新、模块化的 Flarum 2.0 主题，支持品牌定制、暗色模式以及多种页面布局选项。

## 功能特性

### 品牌定制
- 主色调、头部背景色与文字颜色
- 圆角大小控制
- 卡片样式（圆角 / 扁平 / 阴影）

### 暗色模式
- 可自定义暗色背景、表面色和文字颜色

### 布局优化
- 紧凑模式
- 侧边栏开关
- 头像大小控制

### 模块化页面布局（每个模块可独立开关）

| 模块 | 页面 | 说明 |
|------|------|------|
| **首页布局** | `/` | 分类卡片网格 + 最近讨论、精选标签、自定义链接 |
| **标签页** | `/tags` | 标签云 + 分类卡片及热门帖子标题 |
| **帖子列表页** | `/t/{tag}` | 博客风格帖子列表，支持摘要、点赞、回复数、排序和无限滚动 |
| **帖子内容页** | `/d/{id}` | 时间线布局，侧边栏（楼层跳转、参与者、事件）、楼层编号、CSS 参数微调 |

## 安装

```bash
composer require yannisme/melon
```

然后在 Flarum 管理后台启用扩展。
